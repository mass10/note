##### Package Control

https://packagecontrol.io

##### IMESupport

デフォルトでは日本語入力のカーソル位置がおかしいのでこれを解消するやつ。

`Ctrl` + `Shift` + `p` から `Install Package` を選択して `IMESupport` を検索。

##### ConvertToUTF8

UTF-8 以外の日本語文字コードを扱えるようにするやつ。

`Ctrl` + `Shift` + `p` から `Install Package` を選択して `ConvertToUTF8` を検索。
